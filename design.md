# AegisML Cyber Command Center Design System

## MISSION STATEMENT
Completely rebuild the UI into a cyberpunk fintech war room — something between a Tony Stark HUD, a neon-drenched Tokyo arcade, and a high-security bank's fraud ops center.

Design language: LIQUID GLASS + DEEP NEONISM + CLAYMORPHISM AT MAXIMUM INTENSITY.
Every surface breathes. Every metric pulses. Every chart is alive.

## COLORS
- Background base: #03050f
- Background layer 2: #070d1a
- Background layer 3: #0c1428
- Glass surface: rgba(255, 255, 255, 0.04)
- Glass border: rgba(255, 255, 255, 0.10)
- Glass highlight: rgba(255, 255, 255, 0.18)

- Neon Cyan: #00f5ff (primary accent)
- Neon Orange: #ff6b2b (fraud / alert color)
- Neon Violet: #b44fff (secondary accent)
- Neon Green: #00ff88 (safe / clear status)
- Neon Red: #ff2d55 (critical alert)
- Neon Gold: #ffd60a (suspicious tier)
- Neon Pink: #ff3cac (highlight accent)

## TYPOGRAPHY
- Display font: 'Orbitron'
- Body font: 'Rajdhani'
- Mono font: 'JetBrains Mono'

## COMPONENTS
**Liquid Glass Card**:
- Background: rgba(255, 255, 255, 0.04)
- Border: 1px solid rgba(255, 255, 255, 0.10)
- Border Radius: 24px
- Backdrop Filter: blur(24px) saturate(180%)
- Box Shadow: 0 8px 32px rgba(0, 0, 0, 0.6)

**Clay Elements (Badges/KPIs)**:
- Background: linear gradients (e.g., #00f5ff to #0088ff)
- Border Radius: 20px
- Glowing drop shadows matching the accent color

**Neon Glow Text**:
- High contrast, bright neon colors (Cyan, Orange, Violet)
- Layered text-shadows for a glowing effect (e.g., 0 0 10px #00f5ff, 0 0 30px #00f5ff80)
